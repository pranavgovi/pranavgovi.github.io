Research work


Early Alzheimer Detection through speech analysis and Vision transformer approach

Approach: Log mel spectrogram with vision transformer

This research work intends to identify Alzheimer's disease through spontaneous speech. This was published at the International Conference "SPELLL 2022".


Efficiently embedding temporal knowledge graphs for link prediction

Approach: Temporal random walk and temporal proximity

This research work intends to capture the dynamic changes in the knowledge graph for link prediction as a downstream task.




Backtracking Algorithm with Optical Character Recognition for Solving

Approach: CNN based OCR, recursive backtracking algorithm

This work aims to detect blurred and handwritten sudoku puzzles using OCR and solve them using a backtracking algorithm.This was published at the International Conference 'ICCCEM 2022'




Analysing time complexites of codes generated by Chatgpt

Approach: Third party API (codepal) , Regex

This work aims to analyze the time complexities of the code found in the conversations between the developer and ChatGpt. Over 1000 codes were analyzed.
